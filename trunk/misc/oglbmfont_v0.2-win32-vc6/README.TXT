
BMFont OpenGL version 0.2

https://sourceforge.net/projects/oglbmfont

BMfont OpenGL implementation. Integrate any TrueType font converted to a bitmap with this lightweight library. Fit to use with AngelCode's BMFont tool.

Before using this source code, you have to read and accept LICENSE.TXT

There is no documentation at the moment, see the example program BMFtest.c
